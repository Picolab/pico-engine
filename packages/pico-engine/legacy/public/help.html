Pico Engine Help

Managing Picos

Visit the My Picos page to see your owner pico. On the My Picos page you can create a hierarchy of picos.

Managing rulesets

To create your own rulesets, use the Engine Rulesets page.

Reset database

If you wish to start over, remove the database.

  1. Stop your HTTP server (use Ctrl-C).
  2. Use rm -r ~/.pico-engine/db to clear the database. This action will remove all picos, and is not reversible.
  3. Restart the HTTP server (use pico-engine) which will re-create the database, and create a new owner pico.
